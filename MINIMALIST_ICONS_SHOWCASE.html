Принципы Дизайна

●
Рафинированная Простота
Круглая форма с концентрическими кольцами. Минимум деталей — максимум выразительности. Каждый элемент функционален.
✦
Сложные Красные
Не простой #FF0000, а палитра burgundy/vermillion с радиальными градиентами для глубины и объема.
○
Визуальное Разделение
Внешнее кольцо с переменной прозрачностью предотвращает слияние маркеров при зумировании карты.
◐
Тонкие Тени
SVG фильтры создают мягкие, четкие тени (feGaussianBlur 1.2px) для легкого, но заметного объема.
↗
Деликатные Анимации
Пульсация колец, shimmer highlights, number pop — все с кривыми cubic-bezier(0.34, 1.56, 0.64, 1).
⊕
Идеальная Читаемость
Номера маршрутов — DM Sans 700, 14px. Точка для обычных маркеров — 2.5px радиус с 90% opacity.

Технические Детали

Структура: Три концентрических круга — внешнее кольцо (r=20), основной круг (r=15), внутренний highlight (r=6)

SVG Фильтры: Радиальные градиенты + Gaussian blur shadow (1.2px) + Offset (0, 1px)

Анимации: FadeIn (0.35s) → Ring Pulse (2.5s loop) → Shimmer (1.2s loop) → Number Pop (0.3s)

Типографика: DM Sans 700, letter-spacing: -0.3px для номеров

Состояния: data-state атрибут управляет анимациями через CSS селекторы
